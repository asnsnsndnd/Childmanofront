🧒 Child Management Frontend

This is the frontend application for managing child registrations and parent information. Built with React, 
Tailwind CSS, and RTK Query. It connects to a Node.js + Express backend and MongoDB Atlas for data storage.

🚀 Features

Multi-step child registration form

Parent and child information management

File upload support (images/documents)

Responsive design with Tailwind CSS

Light/Dark mode support

Real-time updates using Socket.io (if backend enabled)

🛠️ Tech Stack
Layer	Technology
Frontend	React, Tailwind CSS, FontAwesome
State	Redux Toolkit, RTK Query
Forms	react-hook-form
Notifications	react-toastify
HTTP Requests	axios
Version Control	Git, GitHub
